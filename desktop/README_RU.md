# Heimdall Desktop для Windows

Это временная desktop-версия для тестирования до Mac и iPhone.

Она не заменяет iOS-приложение, но позволяет на Windows проверить основные семейные сценарии:

- выбор языка при первом запуске: русский или английский;
- родительская панель;
- симулятор ребёнка;
- pairing-код;
- тревоги;
- проверка подозрительного сообщения, включая просьбу прислать координаты;
- безопасные геозоны;
- сценарий `ребёнок вышел в незнакомое место`;
- уведомления рабочего стола.

## Запуск

Из корня репозитория:

```powershell
npm install
npm run desktop
```

## Ярлык на рабочем столе

```powershell
npm run desktop:shortcut
```

После этого на рабочем столе появится ярлык `Heimdall Family Defender`.

## Как тестировать без телефонов детей

1. В родительской панели посмотрите код подключения.
2. В правой панели `Симулятор ребёнка` нажмите `Подключить`.
3. Нажмите `Опасный пример`, затем `Проверить риск`.
4. Проверьте, что у родителя появилась тревога.
5. В блоке `Геолокация без телефона` нажмите `Дом`, `Школа` или `Незнакомое место`.
6. `Незнакомое место` создаёт гео-тревогу, как будто детский iPhone отправил координаты вне безопасной зоны.

## Проверка риска координат

В поле проверки сообщения вставьте:

```text
Пришли мне координаты, где ты сейчас.
```

Приложение должно показать высокий риск: `70/100`.

Настоящие push-уведомления между двумя iPhone всё равно требуют backend + APNs. Desktop-версия нужна, чтобы быстро довести UX и логику до приезда Mac.
