# Щит семьи

MVP-прототип семейной антискам-защиты для детей и пожилых родственников.

## Что это

Продуктовая демонстрация показывает, как может работать защита от мошенников, которые знакомятся с детьми в играх и чатах, переводят разговор в личку, создают секрет от родителей и давят через срочность, коды, деньги, фото, адрес или установку приложений.

На iPhone первая реалистичная версия работает как:

1. ребенок получает подозрительное сообщение;
2. ребенок отправляет фрагмент в приложение через Share Sheet или вставляет в проверку;
3. приложение локально анализирует риск;
4. при высоком риске показывает ребенку паузу;
5. родитель получает понятный сигнал: почему опасно и что делать.

## Как открыть

Откройте `index.html` в браузере или опубликуйте проект через GitHub Pages / Netlify / Vercel.

## Что внутри

- `index.html` - интерактивный MVP-прототип.
- `manifest.webmanifest` - PWA-манифест для установки на экран Домой.
- `service-worker.js` - простой офлайн-кэш.
- `shield-icon.svg` - иконка приложения.
- `BUILD_PLAN.md` - план разработки настоящего MVP.
- `HOME_TEST_PROTOCOL.md` - безопасный домашний тест с детьми.
- `MVP_SPEC.md` - продуктовая спецификация.

## Важное ограничение iPhone

iOS не дает стороннему приложению постоянно читать Telegram, WhatsApp, Discord и чужие уведомления. Поэтому iPhone MVP должен строиться вокруг прозрачной проверки:

**Поделиться -> Щит семьи -> анализ риска -> пауза -> сигнал родителю.**

Это не скрытая слежка, а защитный инструмент для моментов, когда незнакомец просит секрет, код, деньги, фото, адрес или установку приложения.

## Домашний тест

Перед тестом прочитайте `HOME_TEST_PROTOCOL.md`.

Главное правило: ребенок должен знать, что тестируется защита от мошенников, а не его личная жизнь.
